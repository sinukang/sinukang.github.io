New Eden

자율 에이전트 3D 샌드박스 게임 | 2025.04.14 ~ 2025.05.30

프로젝트 개요

자율 에이전트를 관찰하고, 플레이어(신)로서 시뮬레이션에 개입할 수 있는 3D 샌드박스 게임입니다. 에이전트는 로컬 AI 모델을 기반으로 자신과 환경을 인식·판단하며, 장기·단기 기억을 축적하여 인간과 유사한 의사결정을 수행합니다.

Unity3D (C#)
Python + FastAPI
Ollama
Gemma 3 SLM
JSON

구현 사항

  • 로컬 AI 기반 에이전트 인지·상호작용
    • Gemma 3 SLM 모델을 Ollama 로컬 서버에서 구동
    • 주변 에이전트 및 환경(다른 NPC, 오브젝트 등) 상태 인식
  • 플레이어 개입 메커니즘
    • 자연어 또는 프리셋 명령어로 특정 에이전트에 지시 전달
    • 음식, 가구, 번개, 생물체 등 오브젝트 배치·제거를 통한 환경 조작
  • 장기·단기 기억 저장
    • 단기 기억: 하루 단위 관찰·행동·피드백 로그
    • 장기 기억: 중요 사건·반복 경험 메타데이터 축적
    • JSON 파일로 저장·로딩하여 게임 구조에 반영

기술 스택 세부

  • Unity 엔진(C# 스크립팅): 에이전트 동작 로직, 환경 오브젝트 관리
  • Ollama + Gemma 3 모델: 4비트 양자화, 로컬 추론 서버 구축
  • Prompt 관리 모듈: 반응 일관성 보장을 위한 JSON 템플릿, 예외 상황 핸들러 포함
  • 메모리·환경 저장: JSON 기반 I/O 레이어, 버전 관리 및 장기·단기 분리
  • 통신: Unity ↔ Ollama RPC (HTTP/WebSocket)

담당 역할

  • AI 팀장 - AI 모델 파이프라인 설계 및 로컬 서버 연동 관리
  • Reflection, Plan, Action, Feedback 데이터 플로우 아키텍처 수립
  • 고급 프롬프트 엔지니어링 및 예외 핸들링 룰 구현

프로젝트 성과

  • 완전 자율 시뮬레이션 월드 구현: 플레이어 개입 없이도 에이전트들이 자체 판단·행동
  • 제로 API 비용: 로컬 SLM 모델 연동으로 외부 의존성 제거
  • 고도화된 메모리 구조: 논문 기반 기억 계층 설계로 에이전트의 장기 학습 가능
  • 효율적 멀티 에이전트 아키텍처: 대규모 에이전트 동시 구동 시 성능 저하 최소화

프로젝트 리뷰 및 회고

프로젝트 중간까지 SLM을 사용하면서 정성평가같은 보고서를 작성하지 않았던 점이 아쉬웠습니다. 유사한 메모리를 불러올 때 부정적인 피드백의 메모리는 유사도를 따로 검사하게끔 하는 로직을 짰던 점이 기억에 남았습니다.

일이 몰려서(AI팀장, PM, AI 업무) 중간에 일이 진행이 되지 않았지만 팀장으로써 어떤식으로 팀원들에게 일을 분배해야 하는지, 팀원으로써도 어떻게 일을 해야 팀에 도움이 되는지 알 수 있었습니다.
