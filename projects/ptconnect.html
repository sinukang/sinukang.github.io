PT-Connect

PT 플랫폼 프로젝트 | 2023.12.19 ~ 2024.01.17

프로젝트 개요

PT-Connect는 PT 트레이너와 회원을 연결해주는 플랫폼 서비스입니다. 트레이너는 자신의 프로필과 스케줄을 등록하고, 회원은 원하는 트레이너를 검색하여 PT 서비스를 예약할 수 있습니다.

Java
Spring Framework
MySQL
JavaScript
Bootstrap
AWS

주요 기능

  • 회원 및 트레이너 회원가입/로그인
  • 트레이너 프로필 및 스케줄 관리
  • PT 예약 시스템
  • 결제 시스템 연동
  • 리뷰 및 평점 시스템
  • 관리자 대시보드

담당 역할

  • 팀장으로서 프로젝트 일정 관리
  • 시스템 아키텍처 설계
  • 회원 및 트레이너 정보관리 기능 구현
  • Spring Interceptor 활용 사이트 보안 개발
  • AWS 서버 배포 및 관리

성과 및 배운 점

PT-Connect 프로젝트를 통해 Spring Framework의 다양한 기능들을 활용하는 방법을 익혔습니다. 특히 Spring Security와 Interceptor를 활용한 보안 시스템 구현에 중점을 두었으며, RESTful API 설계 및 구현 역량을 강화했습니다. 또한 팀장으로서 일정 관리 및 팀원들과의 협업 능력을 기를 수 있었습니다.
